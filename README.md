## Installation
```bash
git clone https://github.com/nachiketkanore/realtime-document-collaboration.git
cd realtime-document-collaboration
npm install

# Run server
cd server
npm run devStart

# Start react app
cd ../client
npm start
```

Now the application will be running on `http://localhost:3000/`
